Was ist Angular?

von Malin Schultz



Definition Was ist Angular?

- TypeScript basiert

-JavaScript Framework

- Hierarchie von Komponenten statt Controller

- Mobile First

- Modularität

 Historie Wo kommt Angular her?

- als Nachfolger von AngularJS

- entwickelt von Google

- first release 2010 als AngularJS

- aktuelleste stabile Version Angular9 im Februar 2020

 Funktionalität Wie funktioniert Angular?

- Komponenten:

- erhält eine Eingabe und ändert das Verhalten passend

- macht das Wiederverwenden von Code einfach

- sind directives, "Markierungen auf DOM Elementen"

- es werden die UI-Teile getrennt in HTML-Tags und
in deren Verhalten in Java- bzw. Type-Script-Code

 Angular vs Vue.js Was ist besser?

- Angular ist älter

- Angular ist länger in großen Projekten im Gebrauch

- Vue.js ist schneller leichter zu lernen

- mit Angular kann ein tieferes Verständnis von Frontend-Funktionalität erreicht werden

- Funktionsweise der Komponeten ist ähnlich

- Integration anderer Bibliotheken ist mit Vue.js leichter

 Angular vs React Was ist besser?

- Google vs Facebook

- React kombiniert UI und Verhalten

- da React kein vollständiges Framework ist, müssen für mehr Funktionen weitere Bibliotheken genutzt werden

- Einrichtung und Erlernen ist bei React leichter

- mit React geschriebener Code kann leicht in andere Frameworks eingebunden werden
